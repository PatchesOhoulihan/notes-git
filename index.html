git clone

With git clone it's possible to duplicate either a remote or local repository

Syntax: git clone URL

EX: git clone https://github.com/PatchesOhoulihan/webdevelopment-skeleton.git

git Remote

Syntax: git remote

git remote without parameter is used to show the name of the remote repository of your local repository.

git remote -v for more output(The complete URL), use the -v Flag.

git remote add

is used to add a remote repository. If you don't have remote repository ist common to name it origin

git remote add origin https:\\....

git push

Use git remote push

Copy your local repository to a remote one

Syntax: git remote push "reponame" "branchname"

EX: git remote push origin master

git pull

Syncronize the local repository with the the remote repository

Syntax: git pull 'repoURL' 'branchname'

EX: git pull origin master
