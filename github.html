Forking a Project

Forking is only used in the context of github.

  1. Go to the Repository on github which you like to Fork
  2. Press the Fork button in the right upper corner to create a copy of the repo. on your on profile
  3. Go to your Profile page
  4. select and open the project you just forked
  5. Clone the project
  6. If you want to push your changes back to the repo copy you just made. There is no need to add a remote repo github already added this for you.(run git remote -v if you don't want to trust on that blindly)
  7. Now you can work with the project as if you're the one who created it.

Colabroating on one Repository

If you want to work together on a Project it's nessecary to add the user you want to collabroate with to your repository.
